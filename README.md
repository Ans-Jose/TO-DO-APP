#  To-Do App  

  Hi! This is my simple **To-Do App** project.  
  It helps users add, complete, and delete tasks easily.  

## Features
  - Add new tasks  
  - Mark tasks as completed  
  - Delete tasks  

##  Technologies Used
  - HTML  
  - CSS  
  - JavaScript  

##  How to Run
  1. Download the code  
  2. Open `index.html` in your browser
